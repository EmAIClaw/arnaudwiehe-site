# Event Logos

This folder should contain logos of events where Arnaud has spoken.

## Required Logos

| Event | Filename | Notes |
|-------|----------|-------|
| Economist Impact | `economist-impact.png` | economist.com |
| Gartner/Evanta | `gartner.png` | gartner.com |
| GITEX | `gitex.png` | gitex.com |
| World Summit AI | `world-summit-ai.png` | worldsummit.ai |
| IKEA | `ikea.png` | ikea.com |
| Heliview | `heliview.png` | heliview.com |
| TechEx | `techex.png` | techex.se |

## Notes
- Logos should be PNG or SVG format
- White or transparent backgrounds preferred
- Recommended size: 200-400px wide
- You can usually find logos on the event's Press/Media page
